FAST Paper List in last 5 years

13 Feb. 2014 - Lubbock

FAST (USENIX Conference on File and Storage Technologies) is a tremendous successful conference in system field. It has been considered as the best system conferences with SOSP and OSDI. FAST has a small community and only accepts a very small number of papers each year. In this post, we will go through all the papers received in last 5 years (2008 - 2013) to see all these successful research groups and works. Although FAST is a conference concentrating storage technologies, it still way out of my knowledge circle to read all of them. So, here, we only list papers about the storage performance, storage devices, and caches, eliminating the failures, CPUs, networks, model checking, power, and provisioning.
